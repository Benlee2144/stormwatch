⚡ About StormWatch

StormWatch is a real-time camera monitoring platform that aggregates over 20,000 live camera feeds from across the United States into a single, dark-themed command center.

What It Does

StormWatch provides instant visual access to traffic cameras, weather stations, skyline views, and airport feeds across 39 states. Whether you're tracking a storm system moving across the plains, checking highway conditions before a road trip, or just watching a city skyline at sunset — StormWatch puts eyes everywhere.

How It Works

We aggregate publicly available camera feeds from state departments of transportation, municipal traffic systems, and public webcam networks. Camera locations are plotted on an interactive map with real-time clustering for fast performance even with thousands of markers.

Each camera feed links directly to its source — we don't proxy or store video. This means you always get the freshest, most current view available.

Tech Stack

StormWatch is built as a lightweight static site for maximum speed:

  • 🗺️ Leaflet.js — Interactive mapping with marker clustering
  • 🎨 Custom Dark UI — Built from scratch, no framework bloat
  • 📡 20,000+ Cameras — Aggregated from public DOT and webcam sources
  • ⚡ Static Generation — Fast loading, no server required
